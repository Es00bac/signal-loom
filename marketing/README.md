## Anti-shadowban guardrails (read before any post)

- **One community at a time.** Never paste the same text into many subs the same day.
- **Follow each sub's self-promo rule** (most have one; many only allow it on certain days/threads).
- **Value first.** Lead with the thing made *with* the app or a genuine "I built this, here's how,"
  not "buy my app." Show, don't sell.
- **Be present.** Reply to every comment for the first 2 hours; engagement beats reach.
- **No mass-DMing, no copy-paste spam, no fake accounts.** Slow and real outranks fast and spammy.
- **Space it out.** A few good posts a week, not 20 in a day.

## Legit automation stack (you connect your own accounts)

- **X/Twitter:** Buffer or Typefully (free tiers) — bulk-load the queue I produce, schedule it.
- **Reddit:** post **manually** (Reddit punishes automation hardest); use the drafts + the timing notes.
- **Discord:** native Scheduled Events + the welcome/AutoMod config in the blueprint.
- **Cadence:** I run on a schedule and hand you the next batch — you approve + queue.
