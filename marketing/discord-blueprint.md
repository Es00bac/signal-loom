# Discord server blueprint — Signal Loom

You create the server (Discord → **＋** → *Create My Own* → *For a club or community*); this is
everything to put in it. ~10 minutes. Doing it yourself keeps it tied to your account, ownership,
and phone verification — which is exactly what you want.

## Server identity
- **Name:** Signal Loom
- **Icon:** `release/play-store/graphics/icon-512.png`
- **Community features ON** (Server Settings → Enable Community) → unlocks announcement channels,
  AutoMod, welcome screen, insights.
- **Verification level:** Medium (must be a verified Discord member for 5 min). **AutoMod:** on
  (block spam links + mention spam).

## Channels

**▸ START HERE**
- `#welcome` — read-only; rules + the welcome message below; reaction-role to enter.
- `#announcements` — read-only, *announcement* type; releases + launch news.
- `#get-the-app` — read-only; the Play test link / direct buy link + install steps.

**▸ COMMUNITY**
- `#general` — main chat.
- `#introductions` — "what do you make?"
- `#showcase` — work made with Signal Loom (seed this yourself before inviting anyone).
- `#feedback-and-ideas` — feature requests; pairs with the in-app roadmap.
- `#help-and-support` — questions/bugs; your direct support channel (Play requires a contact path —
  this helps).

**▸ WORKSPACES** (tips per tool)
- `#flow` · `#image` · `#paper-and-comics` · `#video`

**▸ BEHIND THE SCENES**
- `#dev-log` — build-in-public posts (mirror your X). People follow *founders*, not apps.
- `#roadmap` — what's next; let people vote with reactions.
- `#off-topic`

**▸ VOICE** — `General` VC (optional; for the occasional live build/Q&A).

## Roles
- `@Founder` (you) — color, hoisted.
- `@Closed Tester` — your 12; give them a private `#tester-lounge` (two birds: tester recruiting +
  community seed).
- `@Early Supporter` — anyone who bought it; a small perk role (recognition is free retention).
- `@Member` — default after accepting rules (reaction role).

## #welcome copy (paste in)
> **Welcome to Signal Loom** — a local-first generative-media studio: node-graph **Flow**, a layered
> **Image** editor, comic/print **Paper** layout, and a **Video** timeline. One project, your own AI
> keys, bought once.
>
> 👋 Say hi in **#introductions**, show what you make in **#showcase**, and grab the app in
> **#get-the-app**. Building in public in **#dev-log** — feature ideas welcome in
> **#feedback-and-ideas**.
>
> React with ✅ to enter.

## Rules (paste in #welcome)
1. Be decent. No harassment, hate, or NSFW.
2. Keep it on-topic per channel; no spam or unsolicited DMs.
3. Self-promo only in #showcase, and only work you made.
4. Respect everyone's keys/costs — no asking others to run paid generations for you.
5. Bugs/abuse → @Founder.

## Seeding plan (don't open an empty room)
1. Post 3–5 of your own pieces in `#showcase` with a one-line "how I made this."
2. Pin a "What is Signal Loom?" message in `#general` (reuse the welcome copy).
3. Write the first `#dev-log` entry (the honest "I'm a solo dev launching this" story).
4. **Invite your 12 closed testers here first** — instant non-empty server + your test cohort.
5. Only then put the invite link in your Reddit/X posts, the website footer, and the app's Help menu.

## Invite
- Create a **permanent, no-expiry invite** for now; set a vanity URL once you hit Discord's level-3
  boost (later). Put it everywhere: site footer, every post, in-app Help → Community.
