# 10. Settings & layout

## Settings

Open **Settings** from the gear in the top bar. The main sections:

- **Providers** — connect your AI provider keys and sign in to Vertex AI. See
  [Providers & API keys](04-providers-and-keys.md).
- **Keyboard Shortcuts** — review and learn the shortcuts.
- **Gamepad Bindings** — Signal Loom can map a game controller to brush and tool controls, useful
  on couch/tablet setups.

Settings are stored on your device.

## Per-workspace menus

Each workspace shows the menus that make sense **for that workspace**. The top bar changes as you
switch:

- **Flow** — Project, Flow, View, Window, Help.
- **Image** — Project, File, Edit, Image, Select, Tools, View, Window, Help.
- **Video** — Project, Edit, Timeline, Keyframes, View, Window, Help.
- **Paper** — Project, File, Edit, Layout, Insert, Tools, View, Window, Help.

The **Window** menu is shared across all four and lets you jump to any workspace
(`Ctrl+1`–`Ctrl+4`). On desktop these appear as a native menu bar; on mobile they fold into an
integrated app menu.

## Panels and layout

Workspaces are built from **dockable panels** (Layers, Brushes, Properties, History, Source Bin,
Bookmarks, and so on):

- **Show/hide** panels and toggle the whole interface with **Tab** for a distraction-free canvas.
- **Save Layout** stores an arrangement; a **Layout** menu offers presets (Reset, Balanced, Focus,
  All Panels) so you can switch between, say, a painting layout and a review layout.
- Adjustment dialogs (Levels, Curves, …) open as independent, dockable windows you can leave open
  while you work.

## Mobile, tablet, and DeX

The same project opens everywhere, but the interface adapts:

- On a **phone**, desktop-only panels are hidden (not crammed) and edge handles replace some rails;
  the canvas stays front and center.
- On a **tablet**, in **Samsung DeX**, or on **ChromeOS / Aluminium OS** desktop mode, you get a
  touch-usable version of the full desktop layout.
