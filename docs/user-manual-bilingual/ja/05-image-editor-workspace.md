# Image Editor ワークスペース

Image Editor ワークスペースは、生成 AI ツールを内蔵したレイヤーラスターおよびベクターエディターです。写真レタッチ、デジタルペイント、合成、Flow・Video・Paper ワークスペース用のアセット準備を目的としています。選択範囲、マスク、レイヤー、調整、カラープルーフ、ベクターパス、履歴とスナップショットによる非破壊編集に対応しています。

本章では、レイアウト、すべてのツール、ブラシエンジン、選択範囲とマスクのワークフロー、レイヤー管理、調整、カラープルーフ、生成操作、履歴、アセット書き出し、PSD 読み込みについて説明します。

## レイアウト

Image Editor ワークスペースは以下のように構成されています。

- **Toolbar** — 左側の縦長ストリップにすべての編集ツールがあります。
- **Viewport** — 中央のキャンバスで画像が表示・編集されます。
- **Dockable Panels** — 右側と下部の Layers、Channels、Paths、History、Adjustments、Brushes、Tool Options パネルです。
- **Generative Fill Bar** — AI 操作用のオプションの下部バーです。
- **Asset Bar** — 保存、書き出し、送信アクション用の下部またはサイドバーです。

パネルは再配置、ドッキング、フロート、非表示が可能です。レイアウトはプロジェクトごとに保存されます。

### ビューポート操作

- **Space + ドラッグ** — キャンバスをパンします。
- **Ctrl++ / Ctrl+-** — ズームイン / ズームアウトします。
- **Ctrl+0** — 画像を表示領域にフィットします。
- **100%** — ピクセルを 1:1 で表示します。
- **ピンチジェスチャー** — タッチデバイスでズームします。

## ツール

Image Editor には、すべてのツールが用意されています。ショートカットは括弧内に示します。

### Move（`V`）

レイヤー、選択範囲、ベクターオブジェクトを移動します。選択範囲がアクティブな場合、Move は現在のレイヤー上の選択ピクセルを移動します。

### Hand（`H`）

画像を変更せずにビューポートをパンします。

### Marquee（`M`）

長方形または楕円形の選択範囲を作成します。`Shift` を押しながら正方形または正円に制約します。`Alt` を押しながら中心から描きます。モード：置き換え、追加、減算、交差。

### Lasso（`L`）

自由な選択範囲を作成します。`Alt` を押しながら多角形 Lasso モードに切り替えて、直線セグメントを描きます。

### Magic Wand（`W`）

似た色の連続した領域を選択します。Tool Options パネルで許容値を調整します。許容値が高いほど、より広い範囲の色が選択されます。

### Brush（`B`）

現在のブラシプリセットでペイントします。Brush ツールは筆圧、傾き、ダイナミクス、ブレンディングに対応しています。ブラシエンジンについては後述します。

### Eraser（`E`）

ピクセルを削除し、レイヤータイプに応じて透明または下のレイヤーを露出します。

### Background Eraser（`Alt+E`）

背景色をサンプリングしながら消去し、前景のエッジを保持します。単色の背景を除去するのに便利です。

### Magic Eraser（`Shift+E`）

似た色の連続領域を 1 クリックで消去します。Magic Wand と Delete を組み合わせたような動作です。

### Clone Stamp（`S`）

ある領域のピクセルをサンプリングし、別の場所にペイントします。`Alt` を押しながらクリックしてサンプルソースを設定します。斑点や小さな物体の除去、物体の複製に便利です。

### Spot Heal（`J`）

近くのテクスチャを自動的にサンプリングして、斑点や小さな傷を除去します。Spot Heal ツールは手動のソースポイントを必要としません。

### Blur（`R`）

上にペイントすることでピクセルを柔らかくします。強度とブラシサイズでぼかし量を制御します。

### Sharpen（`Shift+R`）

局所コントラストを増加させます。アーティファクトを避けるため控えめに使用してください。

### Smudge（`U`）

ピクセルを濡れた絵の具のように押し広げます。色の混ざり合いや絵画的な効果に便利です。

### Dodge（`O`）

画像の領域を明るくします。オプションで範囲を制御します：シャドウ、ミッドトーン、ハイライト。

### Burn（`Shift+O`）

画像の領域を暗くします。Dodge と同じ範囲オプションです。

### Sponge Saturate（`P`）

ペイントした場所の色の彩度を上げます。

### Sponge Desaturate（`Shift+P`）

ペイントした場所の色の彩度を下げます。

### Paint Bucket（`G`）

前景色またはパターンで連続領域を塗りつぶします。許容値で、塗りつぶされる色の似た度合いを制御します。

### Gradient（`Shift+G`）

選択範囲またはレイヤーをグラデーションで塗ります。線形、放射、角度、反射、ダイヤモンドなどのタイプがあります。

### Pen（`Shift+B`）

ベクターパスとシェイプを作成します。パスは選択範囲、マスク、シェイプレイヤーに変換できます。

### Rectangle（`X`）

長方形のシェイプを描きます。`Shift` を押しながら正方形にします。

### Ellipse（`Shift+X`）

楕円のシェイプを描きます。`Shift` を押しながら正円にします。

### Crop（`C`）

トリミング領域を定義し、適用します。Tool Options パネルにアスペクト比、解像度、プリセット選択が表示されます。`Enter` で適用、`Esc` でキャンセルします。

### Text（`T`）

テキストレイヤーを作成します。キャンバス上で直接、または Tool Options パネルでテキストを編集します。フォント、サイズ、色、ストローク、影、配置に対応しています。

### Eyedropper（`I`）

画像から色をサンプリングします。`Alt` を押しながら背景色をサンプリングします。サンプリングされた色は前景色になります。

## ブラシエンジン

ブラシエンジンは WebGL2 と CPU レンダリングの両方をサポートしています。WebGL2 は大きなブラシや複雑なダイナミクスで高速です。WebGL2 が利用できない場合は CPU モードがフォールバックになります。

### ブラシダイナミクス

ブラシは以下に応答できます。

- **Pressure** — スタイラスの筆圧からサイズ、不透明度、フロー。
- **Tilt** — スタイラスの傾きから角度と形状。
- **Velocity** — カーソルの速度からサイズまたは不透明度。
- **Jitter** — サイズ、角度、色、位置のランダム変化。

### 対称

対称を有効にすると、水平、垂直、放射状にストロークをミラーリングします。対称オプションは Brushes パネルにあります。

### プリセット

Sloom Studio には、ラウンド、テクスチャ、スキャター、シェイプブラシのプリセットが同梱されています。Brushes パネルからカスタムプリセットを作成し、設定に保存できます。

### テクスチャ

ブラシはテクスチャ画像を使用できます。テクスチャピッカーで Source Bin からテクスチャを読み込むか、ファイルを読み込みます。

### サイズショートカット

- `[` — ブラシサイズを小さくします。
- `]` — ブラシサイズを大きくします。
- `{` — ブラシの硬度を下げます。
- `}` — ブラシの硬度を上げます。
- Windows では `Alt+右クリックドラッグ`、macOS では `Ctrl+Option+ドラッグ` でインタラクティブにサイズ変更します。

## 選択範囲とマスク

### 選択モード

選択ツールがアクティブなとき、Tool Options パネルにモードが表示されます。

- **New** — 現在の選択範囲を置き換えます。
- **Add** — 現在の選択範囲に追加します。
- **Subtract** — 現在の選択範囲から削除します。
- **Intersect** — 重なっている部分だけを保持します。

### フェザーとアンチエイリアス

- **Feather** — ピクセル半径で選択範囲のエッジを柔らかくします。
- **Anti-alias** — 楕円や Lasso の選択範囲のギザギザを滑らかにします。

### オブジェクト選択

Object Selection ツールは AI を使って画像内のオブジェクトを検出します。オブジェクトの周りに大まかな長方形または Lasso を描くと、Sloom Studio は選択範囲をオブジェクトのエッジに補正します。

### Select and Mask

Select メニューから **Select and Mask** を開き、以下でエッジを調整します。

- エッジ検出半径。
- スムーズとフェザー。
- コントラストとエッジシフト。
- 出力先：選択範囲、レイヤーマスク、新規レイヤー。

### Quick Mask（`Q`）

Quick Mask モードは、画像上に一時的なマスクをペイントします。黒でペイントすると選択範囲から削除し、白で追加します。もう一度 `Q` を押すと選択範囲に戻ります。

### チャンネル

Channels パネルには、カラーチャンネルとアルファチャンネルが表示されます。チャンネルを選択範囲として読み込んだり、選択範囲をチャンネルに保存したりできます。

### レイヤーマスク

レイヤーマスクを追加して、ピクセルを削除せずにレイヤーの一部を隠します。マスクに黒でペイントすると隠し、白で表示し、グレーで部分的な透明にします。

### ベクターマスク

ベクターマスクはパスを使って可視性を定義します。どの解像度でも鮮明に保たれます。パスまたはシェイプからベクターマスクを作成します。

### クリッピングマスク

クリッピングマスクは、1 つのレイヤーのコンテンツを使って、上のレイヤーの表示領域を定義します。レイヤーを右クリックして **Create Clipping Mask** を選択します。

## レイヤー

Layers パネルは画像合成の中心です。

### レイヤータイプ

| タイプ | 目的 |
|------|---------|
| Pixel layer | 標準的なラスターコンテンツ。 |
| Adjustment layer | 非破壊的な色調とトーン調整。 |
| Text layer | 編集可能なテキスト。 |
| Shape layer | ベクターシェイプ。 |
| Smart object / source-linked layer | Source Bin アイテムを参照します。 |
| Group layer | レイヤーを整理します。 |

### 表示と不透明度

目のアイコンをクリックして表示を切り替えます。Opacity スライダーをドラッグして透明度を変更します。

### ブレンドモード

ブレンドモードは、レイヤーが下のレイヤーとどのように相互作用するかを決定します。一般的なモードには以下があります。

- Normal
- Multiply
- Screen
- Overlay
- Soft Light
- Hard Light
- Color Dodge
- Color Burn
- Difference
- Luminosity

### ロック

レイヤーは以下のためにロックできます。

- 透明ピクセル
- 画像ピクセル
- 位置
- すべてのプロパティ

### レイヤーエフェクト

ドロップシャドウ、アウターグロー、インナーシャドウ、ベベル、ストローク、カラーオーバーレイなどのエフェクトを追加します。エフェクトは編集可能なままです。

### レイヤーフィルター

後から調整できるスマートフィルターを適用します。フィルターには、ぼかし、シャープ、ノイズ、レンダリング、スタイル化効果が含まれます。

### グループとラベル

レイヤーを選択して `Ctrl+G` / `Cmd+G` を押すとグループ化されます。複雑なドキュメントを整理するためにカラーラベルを割り当てます。

### 検索とリンクレイヤー

Layers パネルの検索ボックスで、名前や種類でレイヤーを検索します。レイヤーをリンクすると、一緒に移動や変形ができます。

### ソースリンクレイヤー

ソースリンクレイヤーは Source Library のアイテムを参照します。ソースが Flow で再生成されると、レイヤーを更新できます。レイヤーを右クリックして **Update from Source** を選択します。

## 調整

調整は色調とトーンを変更します。Image メニューから破壊的に適用するか、Adjustment レイヤーとして非破壊的に適用できます。

### Brightness / Contrast

全体的な明るさとコントラストの簡易コントロールです。

### Levels

ヒストグラムを使って入力と出力レベルを調整します。黒、白、中間調ポイントの設定に便利です。

### Curves

カーブを使った正確なトーンコントロールです。カラーグレーディングのために個別のカラーチャンネルを調整できます。

### Hue / Saturation

色相をシフトし、彩度を上げ下げし、明度を調整します。特定の色域をターゲットにできます。

### Exposure / Offset / Gamma

露出、オフセット、ガンマの HDR スタイルコントロールです。

### Temperature / Tint

色温度と色相を調整してホワイトバランスを補正します。

### Black & White

チャンネルミキサーコントロールでモノクロに変換します。

### Invert

すべての色を反転します。ネガティブやマスクの作成に便利です。

## カラープルーフ

カラープルーフは、画像が異なる出力条件下でどのように見えるかをシミュレートします。

### グレースケールソフトプルーフ

ピクセルデータを変更せずに、画像をグレースケールでプレビューします。

### CMYK ソフトプルーフ

おおよその CMYK 出力をプレビューします。印刷準備に便利です。Sloom Studio は ICC プロファイルを使用してシミュレーションを行います。

### プルーフインテント

ソフトプルーフのレンダリングインテントを選択します。

- Perceptual
- Relative Colorimetric
- Saturation
- Absolute Colorimetric

ターゲットプロファイルは **Image > Proof Setup** で設定します。

## Generative Fill と AI 操作

Generative Fill バーは、AI 支援編集操作を提供します。

| 操作 | 目的 |
|-----------|---------|
| Inpaint | 選択領域を周囲に基づいて塗りつぶします。 |
| Edit | テキストプロンプトで領域を変更します。 |
| Erase | 物体を除去し、背景を塗りつぶします。 |
| Outpaint | 画像の端を超えてキャンバスを拡張します。 |
| Search Replace | プロンプトを使って 1 つの物体を別の物体に置き換えます。 |
| Recolor | 物体の色を変更します。 |
| Remove Background | 背景を自動的に削除します。 |
| Replace Background | 背景を生成コンテンツで置き換えます。 |
| Relight | 被写体の照明を調整します。 |
| Upscale | AI を使って画像の解像度を上げます。 |

### プロバイダー

生成操作は、**設定 > プロバイダー** で設定されたプロバイダーを使用します。Generative Fill バーでプロバイダーを選択します。推定コストは Usage Bar に表示されます。

### 物体検出マスク

一部の生成操作は、物体検出を使って自動的にマスクを作成できます。画像内の物体を選択し、操作を選択すると、Sloom Studio は検出された物体の周りにマスクを生成します。

### コスト見積もり

各 AI 操作は、実行前に推定コストを表示します。推定はプロバイダー、モデル、画像サイズ、操作タイプに依存します。実際のコストは完了後に Usage Bar に記録されます。

## 履歴、スナップショット、アクション

### History パネル

History パネルには、すべての編集が一覧表示されます。状態をクリックするとその時点に戻ります。履歴状態の数は設定で変更できます。

### スナップショット

スナップショットは、特定の時点でのドキュメント全体の状態を保存します。リスクのある編集の前にスナップショットを作成してください。History パネルからスナップショットを復元します。

### アクション

アクションは、再生用の一連の操作を記録します。アクションを記録し、ショートカットを割り当て、他の画像やレイヤーで再生します。

## アセットバーと書き出し

Asset Bar には、現在の画像に対するクイックアクションがあります。

- **Save to Source Bin** — 画像をプロジェクトの Source Library に追加します。
- **Download** — フラット化されたコピーをディスクに保存します。
- **Export Mask** — アクティブな選択範囲またはマスクを画像として保存します。
- **Send to Flow** — Source Bin ノードまたはノード入力参照を作成します。
- **Send to Video** — 画像を Video の Source Bin に追加します。
- **Import PSD** — Photoshop ファイルをレイヤーとして開きます。

### 対応形式

Image Editor は以下を開いて保存できます。

- PNG
- JPEG
- WebP
- TIFF
- PSD
- AVIF（対応時）

### フリー素材検索

Asset Bar からフリー素材を検索します。結果は Source Bin に読み込むか、キャンバス上に直接配置できます。

## PSD 読み込み

PSD ファイルを読み込んで、可能な限りレイヤー、マスク、ブレンドモード、テキストを保持します。非対応機能はフラット化またはスキップされ、読み込み後にレポートが表示されます。

PSD を読み込むには:

1. **ファイル > 読み込み > PSD** を選択するか、PSD ファイルをキャンバスにドラッグします。
2. 読み込みオプションを選択します：レイヤーをマージ、スマートオブジェクトとして読み込む、すべてのレイヤーを保持。
3. **Import** をクリックします。
