# Video ワークスペース

Video ワークスペースは、ソースメディアからコンポジションを組み立てるノンリニアエディターです。Source Bin、Source Monitor、Program Monitor、マルチトラックタイムライン、Inspector、効果、キーフレーム、キャプション、書き出しプリセットを統合しています。本ワークスペースは、短尺ソーシャルコンテンツ、レビューカット、アーカイブマスター、テキスト・シェイプ・コミック要素を含むモーショングラフィック向けに設計されています。

本章では、コンポジション、モニター、タイムライン編集、効果、音声、シーケンス設定、レンダリング/書き出し、キャプション、FCP XML、モバイルレイアウトについて説明します。

## ドッキング可能なパネル

Video ワークスペースでは、プロジェクトごとにレイアウトを保持するドッキング可能なパネルを使用します。

| パネル | 目的 |
|-------|---------|
| Project Source Bin | 現在のプロジェクト用のメディアを参照・読み込みます。 |
| Source Monitor | ソースクリップをプレビューし、In/Out ポイントを設定します。 |
| Program Monitor | 編集中のシーケンスの現在フレームを表示します。 |
| Inspector | 選択したクリップ、トラック、効果のプロパティを編集します。 |
| Timeline | トラック上にクリップを配置します。 |
| Premiere Parity | Premiere Pro 機能との互換性をチェックします。 |
| Sequence Settings | アスペクト比、解像度、フレームレートを設定します。 |
| Export Preset | レンダリングプリセットを選択・設定します。 |
| Diagnostics | タイムラインエラー、メディア欠落、パリティ問題を報告します。 |

パネルのヘッダーをドラッグして、再配置、ドッキング、フロートを行います。**View > レイアウト初期設定** でレイアウトをリセットできます。

## コンポジション

コンポジションは、Video ワークスペース内の最上位の動画プロジェクトです。トラック、クリップ、効果、シーケンス設定を含みます。プロジェクトには複数のコンポジションを含めることができます。

### コンポジションの作成

新しいコンポジションを作成するには:

1. ワークスペースメニューまたは Source Bin の **New Composition** をクリックします。
2. 開始用シーケンスプリセットを選択するか、空から始めます。
3. コンポジションに名前を付けます。

### スターターシーケンス

スターターシーケンスは、一般的なセットアップを自動的に作成します。例:

- タイトルセーフオーバーレイ付きのソーシャル縦型 9:16。
- 2 本のビデオトラックと 1 本の音声トラックを持つレビュー用 16:9。
- デフォルトトラックを持つ空のタイムライン。

### 履歴

Video ワークスペースは最大 80 件の履歴を保持します。`Ctrl+Z` / `Cmd+Z` で元に戻し、`Ctrl+Y` / `Cmd+Shift+Z` やり直します。履歴は Video ワークスペースにスコープされ、Flow や Paper の編集には影響しません。

## Source Bin

Video ワークスペースの Source Bin には、編集に使用できるメディアが表示されます。以下をサポートしています。

- **メディア読み込み** — 動画、音声、画像、字幕ファイルの追加。
- **種類フィルター** — 動画、音声、画像、シーケンスのみを表示。
- **検索** — 名前、タグ、メタデータでフィルタリング。
- **スター / 折りたたみ** — お気に入りをマークし、グループを折りたたむ。
- **Editor Assets** — あらかじめ作成されたタイトル、シェイプ、トランジション。
- **Paper Storyboard Import** — Paper ワークスペースからストーリーボードフレームを読み込む。

ソースを右クリックすると、プレビュー、配置、トラックに追加、Source Library で表示などのアクションが表示されます。

## Source Monitor

Source Monitor は、ソースクリップをタイムラインに追加する前に個別にプレビューします。

### In/Out の設定

- `I` を押して **In** ポイントを設定します。
- `O` を押して **Out** ポイントを設定します。
- スクラバー上の In/Out ハンドルをドラッグします。
- `Alt+I` / `Alt+O` で In/Out をクリアします。

### 挿入と上書き

In/Out を設定した後:

- `,`（コンマ）を押して、再生ヘッド位置にクリップを**挿入**し、後続のクリップを右に押し出します。
- `.`（ピリオド）を押して、再生ヘッド位置にクリップを**上書き**し、そこにあるものを置き換えます。
- **Add to Track** をクリックすると、再生ヘッド位置を変えずに対象トラックにクリップを配置します。

## Program Monitor と Stage

Program Monitor は、タイムラインの現在フレームを表示します。2 つのモードがあります。

- **Stage** — インタラクティブな変形ハンドル、クリップ境界、ステージオブジェクトを表示します。
- **Rendered** — 最終的なレンダリング出力を表示し、効果や合成をプレビューするのに便利です。

### インタラクティブなクリップ変形

Stage モードでクリップを選択すると、変形ハンドルが表示されます。以下が可能です。

- ドラッグして移動。
- 角をドラッグしてスケール。
- バウンディングボックスの外側をドラッグして回転。
- `Shift` を押しながら比率を固定。
- `Alt` を押しながら中央を基準に変形。

### コミックしっぽのドラッグ

コミックの吹き出しや思考の吹き出しを含むクリップには、しっぽハンドルが表示されます。しっぽをドラッグして、フレーム内のキャラクターやオブジェクトを指すようにします。

### ステージオブジェクト

ステージオブジェクトは、テキスト、シェイプ、コミックパネルなどのオーバーレイです。クリップのように動作しますが、Video ワークスペース内で作成されるため、外部から読み込まれるものではありません。ステージオブジェクトを選択すると、Inspector でそのコンテンツを編集できます。

### フィットモード

シーケンスフレームの表示方法を制御します。

- **Fit** — モニター内に全体が収まるようにスケールします。
- **Fill** — モニターを埋めるようにスケールし、必要に応じて端をトリミングします。
- **100%** — ピクセルを 1:1 で表示します。

### クイックコントロール

Program Monitor の近くに、よく使うアクションのクイックコントロールが表示されます。

- 再生 / 一時停止
- 1 フレーム戻る / 進む
- In / Out の設定
- ステージオーバーレイの切り替え
- 現在フレームのスナップショット

## タイムラインツール

タイムラインツールバーには、編集ツールが含まれます。

| ツール | ショートカット | 目的 |
|------|----------|---------|
| Select | `V` | クリップを選択して移動します。 |
| Cut / Razor | `C` | 再生ヘッド位置でクリップを分割します。 |
| Slip | `Y` | クリップの In/Out を変更せずにタイムライン上の持続時間を維持します。 |
| Hand | `H` | タイムライン表示をパンします。 |
| Snap | `S` 切り替え | クリップと再生ヘッドをエッジやマーカーにスナップします。 |

## トラック

タイムラインには以下があります。

- 映像、画像、テキスト、シェイプ、効果用の **4 本の映像トラック**。
- 音声クリップとナレーション用の **4 本の音声トラック**。

トラックは以下の状態にできます。

- **Locked** — 誤った編集を防ぎます。
- **Collapsed** — スペースを節約するためトラック詳細を非表示にします。
- **Resized** — トラック境界をドラッグして高さを変更します。
- **Targeted** — 対象トラックが Source Monitor からの挿入と上書きを受け取ります。

### オーバーレイトラック

1 本の映像トラックをオーバーレイトラックとして指定できます。オーバーレイクリップは、ブレンドモードとアルファチャンネルを使って上に合成されます。ロゴ、ロワーサード、コミック吹き出しに使用します。

## 編集操作

### ドラッグアンドドロップ

Source Bin からソースをトラックにドラッグします。目的の時間位置にドロップします。ドラッグ中に `Alt` を押すと複製できます。

### 移動とトリム

- クリップをドラッグして、時間方向または別のトラックに移動します。
- 左端または右端をドラッグして、In または Out ポイントをトリムします。
- トリム中に `Shift` を押すと、リップルトリムになります。

### 分割

分割したい位置に再生ヘッドを置き、`C` を押すか Cut ツールをクリックします。クリップは 2 つの独立したクリップに分かれます。

### リップルトリム

リップルトリムは、タイムラインの持続時間を自動的に調整します。

- `Q` を押して、クリップの先頭を再生ヘッド位置までリップルトリムします。
- `W` を押して、クリップの末尾を再生ヘッド位置までリップルトリムします。

### ロール編集

`E` を押してロール編集モードに入ります。ロール編集は、2 つの隣接するクリップ間の境界を調整し、総持続時間は変更しません。

### スリップ編集

Slip ツール（`Y`）を使ってクリップをドラッグすると、タイムライン上の位置や持続時間を変えずに、ソースのどの部分を表示するかを変更できます。

### ギャップの埋め合わせ

クリップ間の空きスペースを右クリックして **Fill Gap** を選択すると、後続のクリップを左に移動して隙間を閉じます。

### スナップ

`S` でスナップを切り替えます。スナップがオンになっていると、再生ヘッド、クリップエッジ、マーカーが互いにスナップします。スナップ感度は設定で調整できます。

### マーカー

`M` でタイムラインにマーカーを追加します。マーカーには名前と色を付けられます。メモ、チャプターポイント、同期基準に便利です。

### タイムラインのズーム

`+` と `-`、または `Ctrl` を押しながらスクロールしてタイムラインを水平方向にズームします。個別のトラックをリサイズして垂直方向にズームします。

## トランスポートと JKL シャトル

トランスポートコントロールは、再生ヘッドを移動し、タイムラインをプレビューします。

| キー | アクション |
|-----|--------|
| `Space` | 再生 / 一時停止 |
| `J` | 逆方向シャトル（繰り返し押すと高速化） |
| `K` | シャトル停止 |
| `L` | 順方向シャトル（繰り返し押すと高速化） |
| `Left Arrow` | 1 フレーム戻る |
| `Right Arrow` | 1 フレーム進む |
| `Home` | 先頭へ移動 |
| `End` | 末尾へ移動 |

`J` または `L` を押しながら `K` を押すと、シャトルが 1 段階低速になります。

### 不透明度と音量の微調整

クリップを選択している状態で:

- `Ctrl+Shift+Up` / `Ctrl+Shift+Down` で映像クリップの不透明度を微調整します。
- `Ctrl+Shift+Left` / `Ctrl+Shift+Right` で音声クリップの音量を微調整します。
- `Alt` を押しながらより小さな増分で微調整します。

## キーフレームとアニメーション

多くのプロパティをキーフレームでアニメーションできます。

- 位置、スケール、回転、不透明度。
- 効果パラメータ。
- 音声の音量とパン。

キーフレームを追加するには:

1. 目的の時間に再生ヘッドを移動します。
2. Inspector でプロパティの横にあるダイヤモンドをクリックします。
3. プロパティ値を変更します。
4. 再生ヘッドを移動して繰り返します。

キーフレームはリニア、イージング、ステップにできます。タイムライン上でキーフレームをドラッグしてタイミングを調整します。

## 効果

Inspector には、選択したクリップの Effects セクションがあります。

### クリップフィルター

クリップフィルターは、以下のような画像調整を適用します。

- 明るさ / コントラスト
- 彩度
- 色相回転
- ぼかしとシャープ
- カラーグレーディング LUT

### クロップ

Crop 効果はクリップの端を取り除きます。キーフレームでアニメーション化して、露出効果を作ることができます。

### クロマキー

Chroma Key 効果は、通常は緑または青の色背景を除去します。Inspector で許容範囲、エッジの柔らかさ、スピル抑制を調整します。

### ストロークとアウトライン

クリップ、テキスト、シェイプの周囲に境界線を追加します。色、幅、角のスタイルを設定します。

### ブレンドモード

ブレンドモードは、クリップが下のトラックとどのように合成されるかを制御します。一般的なモードには Normal、Multiply、Screen、Overlay、Soft Light、Difference があります。

### トランジション

同じトラック上の隣接するクリップ間にトランジションを配置します。利用可能なトランジションには以下があります。

- クロスディゾルブ
- カラーディップ
- ワイプ
- スライド
- フェードトゥブラック

トランジションを Source Bin または Effects パネルからカットポイントにドラッグします。

### 反転

クリップを水平または垂直に反転します。ミラーになった映像の修正や反射の作成に便利です。

## テキスト、シェイプ、コミック

### テキスト

ツールバーまたは Source Bin からテキストを追加します。Program Monitor 上で直接、または Inspector でコンテンツを編集します。オプションには以下が含まれます。

- フォントファミリー、サイズ、太さ。
- 塗り、ストローク、影。
- 配置とトラッキング。
- テキストアニメーションプリセット。

### シェイプ

長方形、楕円、多角形、線を追加します。シェイプには塗り、ストローク、アニメーションを設定でき、マスクに変換できます。

### コミック

コミックツールには以下が含まれます。

- **Panels** — フレームをコミックのコマに分割します。
- **Speech bubbles** — ドラッグ可能なしっぽ付きの吹き出しを追加します。
- **Thought bubbles** — 雲形の吹き出しを追加します。
- **SFX decals** — あらかじめ作成された効果音グラフィック。
- **Captions** — 字幕スタイルのテキストオーバーレイ。

コミック要素はステージオブジェクトであり、他のクリップと同様にアニメーションできます。

## 音声

### 音声クリップ

音声ファイルを音声トラックにドラッグします。十分にズームインすると、タイムラインに波形が表示されます。

### トラック音量

トラックヘッダーのフェーダーでトラック全体の音量を調整します。トラック音量もキーフレームで自動化できます。

### 音量キーフレーム

音声クリップを選択し、Inspector またはタイムラインの音量ラインをクリックして音量キーフレームを追加します。

### フェードとクロスフェード

- **Fade In / Fade Out** — クリップのコンテキストメニューから追加するか、フェードハンドルをドラッグします。
- **Crossfade** — 2 つの音声クリップを重ねて、クロスフェードトランジションを適用します。

### 波形表示

トラックごとに波形表示を切り替えます。波形の解像度はズームレベルに依存します。

### ナレーション生成

ナレーションツールを使って、テキストから音声を生成します。

1. 音声トラックを選択します。
2. トラックメニューから **Generate Narration** を選択します。
3. テキストを入力し、ボイスを選択します。
4. 生成された音声クリップがトラック上に表示されます。

ナレーションは、設定された ElevenLabs またはシステム TTS プロバイダーを使用します。

## シーケンス設定

シーケンス設定は、コンポジションのフレームサイズとタイミングを定義します。

| 設定 | オプション |
|---------|---------|
| アスペクト比 | 16:9、9:16、1:1、カスタム |
| 解像度 | 720p、1080p、4K、カスタム |
| フレームレート | 24、25、30、60 fps |
| ピクセルアスペクト | Square、anamorphic |
| 音声サンプルレート | 44.1 kHz、48 kHz |

スターターシーケンスを選択するとこれらが自動的に設定されます。後から変更するには **Sequence Settings** を開きます。編集後にシーケンス設定を変更すると、既存のクリップがスケールまたはクロップされることがあります。

## レンダリングと書き出し

Export パネルは **ファイル > 書き出し** またはワークスペースツールバーから開きます。

### 書き出しプリセット

| プリセット | 最適な用途 |
|--------|----------|
| Review H.264 | クイックな共有とレビュー。 |
| Social Vertical | 短尺プラットフォーム向け 9:16 コンテンツ。 |
| Archive | ロスレスまたは高ビットレートマスター。 |
| WebM VP9 | Web 再生とオープンソースワークフロー。 |
| GIF | 短いループアニメーション。 |
| ProRes | プロフェッショナルなポストプロダクション交換。 |
| HEVC | 効率的な高品質配信。 |
| PNG/JPEG Sequence | 合成用のフレーム単位出力。 |

### レンダリングバックエンド

Sloom Studio は複数のレンダリングバックエンドをサポートしています。

- **CPU** — どこでも互換性があり、複雑な効果では遅い。
- **WebGL** — GPU アクセラレーション効果で高速。
- **Native** — OS のメディアフレームワークが利用可能な場合に使用。

バックエンドは Export パネルまたは設定で選択します。

### インクリメンタルキャッシュ

レンダリングされたフレームはインクリメンタルにキャッシュされます。タイムラインの一部だけを変更した場合、影響を受けるフレームだけが再レンダリングされます。キャッシュは Export パネルからクリアできます。

### レディネスチェック

レンダリング前に、Sloom Studio は以下をチェックします。

- メディア欠落。
- オフライン効果。
- シーケンス設定の不一致。
- 特定の形式に対するライセンスゲート。

失敗した書き出しを避けるため、警告を修正してください。

### プレビューの保存

**Save Preview** ボタンは、フル書き出しを実行せずに、低解像度のプレビューをレンダリングしてクイックレビューします。

## キャプション

### キャプションの読み込み

SRT または WebVTT ファイルをタイムラインに読み込みます。キャプションは、キャプショントラック上のクリップのようなオブジェクトとして表示されます。

### キャプションの編集

キャプションをダブルクリックして、テキストとタイミングを編集します。Inspector には開始時間、終了時間、スタイルが表示されます。

### キャプションの書き出し

キャプションを SRT または WebVTT として **ファイル > 書き出し > Captions** から書き出します。

## FCP7 XML 書き出し

Sloom Studio は、他のエディターとの交換用に Final Cut Pro 7 XML ファイルを書き出すことができます。書き出しには以下が含まれます。

- クリップと編集。
- トラック構造。
- マーカー。
- 基本的なトランジション。

一部の効果や生成コンテンツは正確に変換されないことがあります。書き出し前に **Premiere Parity** パネルで互換性を確認してください。

## 診断とパリティパネル

Diagnostics パネルはタイムラインの問題を報告します。Premiere Parity パネルは、タイムラインを Premiere Pro の機能と比較し、非対応機能を警告します。

一般的な問題には以下が含まれます。

- ソースメディアの欠落。
- 非対応コーデック。
- FCP7 XML 書き出しで変換されない効果。
- フレームレートの不一致。

## モバイルレイアウト

Android や幅の狭いウィンドウでは、Video ワークスペースが適応します。

- パネルがボトムシートになります。
- タイムライントラックがタッチ用に高くなります。
- JKL シャトルは画面上のトランスポートボタンに置き換えられます。
- 精密編集はロングプレスメニューを使用します。

モバイルでは、高度なドッキングやフローティングパネル機能が制限されます。

## ベストプラクティス

- スケールの予期せぬ変更を避けるため、編集前にシーケンス設定を設定してください。
- 長いタイムラインを整理するため、マーカーを使用してください。
- 編集していないトラックはロックしてください。
- Rendered モードで頻繁にプレビューし、効果の問題を早期に発見してください。
- 繰り返しの書き出しを高速化するため、インクリメンタルキャッシュを活用してください。
- 最終レンダリング前にレビューカットを書き出してください。
