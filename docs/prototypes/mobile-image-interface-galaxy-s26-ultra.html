Phone Image Workspace Shell

This is the proposed phone-specific layout. Desktop, DeX, and tablet keep the current dockable/tabbed floating panels. Phone uses edge drawers and accordion panels.

  • Top: document tabs, plus/new canvas, open image, workspace tabs, quick actions.
  • Left: Source Library / generated assets drawer.
  • Right: dialogs as vertical accordions, not desktop floating tabs.
  • Bottom: assets, brushes, styles, macros.
  • Center: canvas plus draggable Photoshop-style tools palette.
  • Round button: hides/restores all UI, including floating tools.

Sim target: Galaxy Ultra-class phone, QHD+ physical display with a high-density CSS viewport approximation.
