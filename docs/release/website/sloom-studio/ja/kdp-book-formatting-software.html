小説の本文、低コンテンツのジャーナルや手帳、パズルやアクティビティブック、フルカラーの児童書——どれであっても、Sloom Studioはレイアウトから印刷までのパイプラインを、完全に自分のものになるひとつのアプリでカバーします。

原稿から印刷対応まで、ひとつのアプリで

本文をレイアウトするグリッド、ガイド、余白、ガター、断ち落とし、トリムを備えたページ。リフロー可能な本文のために、ページをまたいで流れる連結テキスト。
表紙をデザインするフルラップカバーや背表紙のためのレイヤー式画像エディタ——マスクと範囲選択ツールで、描き、アートを配置し、レタッチし、文字を組みます。
印刷対応の書き出しKDP対応の本文と表紙、埋め込みICCアウトプットインテントを通じた実際のCMYKで適合PDF/X-1a・PDF/X-4、名前付きスポットのSeparation版、加えてCMYKとスポットのスウォッチ。Adobe IDML書き出しは構造的に有効で仕様通り（IDMLインポートはまだ未対応）。
自分のファイルを所有するすべてがあなたの端末上の、普通のプロジェクトファイルです。クラウドへの縛りなし、アカウント不要、完全にオフラインで動作。

Paperで組んだ印刷対応の仕上がり—コマ割り、連結した吹き出し、インスペクタで開いた製版設定。
