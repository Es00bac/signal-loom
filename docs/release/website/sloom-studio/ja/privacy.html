プライバシーポリシー

最終更新：2026年7月5日 · 発行元：Sloom Software

要点だけ言うと。Sloom Studioのアプリはローカルファーストで、 ユーザーアカウントもSloomのサーバーもありません：あなたのプロジェクト、メディア、APIキーは あなたの端末に留まり、アプリはあなたに関する情報を私たちに一切送信しません。あなたの端末を離れる唯一のコンテンツは、 あなたが自分の認証情報でサードパーティのAIプロバイダーに送ると選んだものだけです。唯一の例外が このウェブサイトで、プライバシーに配慮したクッキーなしの訪問数の記録と、あなたが管理できる任意のメール登録機能を提供しています。詳細は以下の通りです。

1. 収集する情報

Sloom Studioアプリを通じては、個人データを一切収集しません。アプリには、私たちにデータを送信するアナリティクス、広告SDK、トラッキング、クラッシュレポートは含まれておらず、Sloomのバックエンドと通信することもありません。 （sloom.studioのウェブサイトについては、セクション1aをご覧ください。）

1a. このウェブサイト（sloom.studio）

アナリティクス。このマーケティングサイトは、ファーストパーティのクッキーを使わない訪問測定を利用しています： 集計されたページビューと、参照元ウェブサイトのホスト名（例：「google.com」）を記録します。クッキー、クロスサイトトラッキング、 広告ネットワーク、サードパーティのアナリティクスサービスは一切ありません。あなたのデータをURLに含めることは決してなく、訪問者のプロファイルを構築することもありません。

メール登録（任意）。登録フォームにメールアドレスを入力することを選んだ場合、そのアドレスをただひとつの目的—— Sloom Studioの更新情報をたまにメールでお送りするため——に、私たち自身のサーバー上に保存します。販売したり共有したりすることはなく、 どのメールにも返信するか、support@sloom.studio宛に ご連絡いただければ、いつでも配信を停止できます。フォームを送信しなければ、私たちがあなたのメールアドレスを持つことは一切ありません。

2. あなたの端末に保存されるデータ

Sloom Studioはアプリが動作するために、以下をあなたの端末上にローカルに保存します。このデータは私たちに送信されません：

  • あなたが入力するプロバイダーのAPIキーと設定。再入力の手間を省くため、アプリのローカルストレージに保存されます。
  • プロジェクト（.sloomファイル）と、取り込みまたは生成したメディア。
  • アプリの設定（レイアウト、ツール、UIの設定）。

プロジェクトを削除する、アプリのストレージを消去する、またはアプリをアンインストールすることで、いつでもこのデータを削除できます。

3. サードパーティのAIプロバイダーに送信するデータ

Sloom Studioは自分のキーを使うツールです。生成、編集、その他のAI操作を実行すると、アプリはその操作に必要な入力 —たとえばあなたのプロンプトのテキストや、選択した画像・動画・音声—を、あなたの端末から、その操作のために選んだAIプロバイダーへ 直接送信し、あなたが提供したAPIキーで認証します。設定によっては、Google（Gemini / Vertex AI）、OpenAIまたは OpenAI互換エンドポイント、Hugging Face、Stability AI、Black Forest Labs、Atlas Cloud、ElevenLabs、その他あなたが設定した 同様のAIプロバイダーが含まれる場合があります。

これらのプロバイダーは独立した企業です。それらの利用はそれぞれの利用規約とプライバシーポリシーに従い、利用料は それぞれのプロバイダーからあなたに直接請求されます。私たちはその関係の当事者ではなく、そのコンテンツのコピーを 受け取ることも一切ありません。コンテンツを送信する前に、各プロバイダーのプライバシーポリシーをご確認ください。これらの転送が発生するのは あなたが起こした操作の結果としてのみであり、あなたが選んだ送信先のみに送られます。Sloom Studioが自発的にあなたのコンテンツを どこかに送ることはありません。

4. 権限

  • インターネット — あなたが設定したAIプロバイダーと通信するため。
  • ストレージ（レガシー、旧バージョンのAndroidのみ） — プロジェクトと書き出したファイルを共有フォルダに 保存するため。最新のAndroid（13以降）ではストレージ権限は一切要求されません。

メディアを取り込む際は、標準のAndroidファイルピッカーでファイルを選びます。これにより、Sloom Studioは選択したファイルのみに アクセスでき—広範な写真・メディア権限は要求されません。すべてのメディアはあなたの端末上で処理されます。

5. お子様のプライバシー

Sloom Studioは一般・成人層向けのプロフェッショナルな創作ツールです。13歳未満のお子様を対象としておらず、 お子様から意図的に情報を収集することはありません。

6. データセキュリティ

あなたのデータはあなたの端末に留まり、私たちは一切保持しないため、Sloom側に侵害されうるデータベースはありません。 AIプロバイダーへのネットワークリクエストは暗号化された（HTTPS）接続で行われます。端末自体の保護（画面ロック、OSの更新、 アクセスできる人の管理）は、あなた自身の責任です。

7. データの削除

すべてのデータはあなたの管理下にあります。削除するには：個々の.sloomプロジェクトを削除する、Android設定で Sloom Studioのアプリストレージを消去する、および/またはアプリをアンインストールしてください。以前AIプロバイダーに送信した コンテンツを削除するには、そのプロバイダー自身の管理機能をお使いください。

8. このポリシーの変更

このポリシーを変更する場合は、上記の「最終更新」日を更新し、新しいバージョンをこのURLに掲載します。

9. お問い合わせ

このポリシーに関するご質問：support@sloom.studio。
