The export formats print actually needs

Amazon KDP, a print-on-demand service, and a traditional printer each want different things. Sloom Studio exports the ones that matter: KDP-ready interiors and covers, conformant PDF/X-1a and PDF/X-4 with real CMYK through an embedded ICC output intent, named spot-color swatches written as real Separation plates, and text embedded as vector curves where supported. You also get a structurally valid, spec-correct Adobe IDML package for InDesign/Affinity Publisher handoff (IDML import is not yet supported), plus CBZ and a webtoon strip for digital. The browser PDF export is a fast RGB proof; the PDF/X path is the press-certified output.

The Paper workspace's live production spec sheet: PDF/X-1a/X-4, real CMYK with embedded ICC, named spot Separation plates, true drop caps, hanging indents, runaround text, and true small caps.
